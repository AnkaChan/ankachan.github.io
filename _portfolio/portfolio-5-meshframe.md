MeshFrame is a header-only mesh processing library with superior efficiency compared to OpenMesh, CGAL, and MeshLab. It supports dynamic mesh editing, runtime properties, and both triangular and tetrahedral meshes. MeshFrame2 serves as the core mesh library in the Gaia Engine.
